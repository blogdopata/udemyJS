Generar un servidor Local con node

  • Todo como administrador
  • Instalar node
  • Instalar : npm install http-server -g
  • http-server [path] [options]
  • Path : indicar la ruta de la carpeta a levantar
  • Option: darle "p" y pone r 8080

http-server C:\Users\victor_caballero1\Desktop\js -p 8080
